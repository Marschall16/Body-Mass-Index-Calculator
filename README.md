# Введение

* Калькулятор индекса массы тела - антропометрический показатель, отражающий соотношение массы и роста человека. Позволяет оценить наличие дефицита, нормального веса, избыточной массы и различных степеней ожирения.

* Основное назначение продукта - предоставить пользователю возможность мгновенно рассчитать свой ИМТ и получить базовую интерпретацию результата в соответствии с классификацией Всемирной организации здравоохранения (ВОЗ).


# Примеры использования

Нужные функции:

1.  * Ввод веса - при клике мыши в поле (Введите ваш вес) пользователь может ввести своё текущее значение веса.
2.  * Ввод роста- при клике в поле (Введите ваш рост) пользователь может ввести свой рост с клавиатуры.
3.  * Запуск расчета - при нажатии на кнопку (Рассчитать ИМТ) программа считывает введенные данные и запускает алгоритм вычисления.
4.  * Отображение числового значения ИМТ - после расчета на экране появляется число индекс массы тела, округленный до десятых.
5.  * Расшифровка результата - в месте с числовым значением выводится текст: (Дефицит массы), (Норма), (Избыточная масса).
6.  * Обработка пустых полей - если пользователь нажмет кнопку (Рассчитать), не заполнив одно из полей, появится всплывающее предупреждение (Invalid).
7.  * Визуальная индикация результата - в зависимости от полученного ИМТ, блок с результатом подсвечивается разным цветом.
8.  * Сброс результатов - при повторном нажатии на кнопку (Рассчитать).
